### What is mind-git?

mind-git implements **CanvasL** - a visual programming language where spatial arrangements of nodes correspond to polynomial algebra over the binary field F₂. The system transforms visual diagrams (Obsidian Canvas files) into executable JavaScript while maintaining formal mathematical guarantees through a 1,400-year chain of mathematical proofs.

### Key Innovation

Unlike traditional visual programming tools that use diagrams as metaphors for code, mind-git's spatial arrangements **are** the mathematical structure. Node positions directly encode polynomial coefficients, graph topology maps to algebraic divisibility, and the observer at coordinates (0,0) serves as the identity element in polynomial algebra.

### Technical Architecture

**Three-Layer System:**

1. **Logos System**: Mathematical engine implementing polynomial algebra, identity chain operations, and Assembly-Algebra Language (AAL) with 11-dimensional graded modal types

2. **Canvas Visual Compiler**: Parser, AST generator, and multi-language code generation (JavaScript, TypeScript, Racket, WebAssembly)

3. **Formal Verification**: Coq proofs ensuring mathematical correctness of all core operations

### Node Classification System

Spatial nodes are classified by text prefixes corresponding to assembly operations:

- `#Activate:` - Linear transformation (JMP/CALL, D0)
- `#Integrate:` - Polynomial addition (ADD/SUB, D1)
- `#Propagate:` - Polynomial shift (SHL/SHR, D2)
- `#Transform:` - Polynomial multiplication (MUL/DIV, D4)
- `#Verify:` - Consensus voting (VOTE, D5)
- `#Store:` - Memory stack (PUSH/POP, D6)
- `#Observe:` - Quantum observation (SYNC, D7)
