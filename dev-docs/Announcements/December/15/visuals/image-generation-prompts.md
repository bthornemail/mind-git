## Quick Generation Commands

### Using DALL-E 3 (via OpenAI API)
```bash
curl -X POST https://api.openai.com/v1/images/generations \
  -H "Authorization: Bearer $OPENAI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "dall-e-3",
    "prompt": "[PASTE PROMPT HERE]",
    "size": "1792x1024",
    "quality": "hd",
    "n": 1
  }'
```

### Using Stable Diffusion (via Stability AI API)
```bash
curl -X POST https://api.stability.ai/v1/generation/stable-diffusion-xl-1024-v1-0/text-to-image \
  -H "Authorization: Bearer $STABILITY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text_prompts": [{"text": "[PASTE PROMPT HERE]"}],
    "cfg_scale": 7,
    "height": 1024,
    "width": 1024,
    "samples": 1,
    "steps": 30
  }'
```

## Post-Processing Recommendations

### For All Images
1. **Optimize file size**: Use TinyPNG or ImageOptim
2. **Add text overlays**: Use Figma or Canva for final text
3. **Ensure accessibility**: Check contrast ratios (WCAG AA minimum)
4. **Test at multiple sizes**: Preview at thumbnail size
5. **Add alt text**: Write descriptive alt text for each image

### Recommended Tools
- **Figma**: Final editing, text overlays, layout adjustments
- **Photopea**: Free Photoshop alternative for touch-ups
- **Remove.bg**: Remove backgrounds if needed
- **Squoosh.app**: Image compression and format conversion
- **Carbon.now.sh**: Code screenshots with syntax highlighting
