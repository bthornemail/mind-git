## Best Practices

### Canvas Design
- **Observer First**: Always place observer at (0,0)
- **Spatial Logic**: Use position to encode complexity
- **Dimensional Progression**: Build from simple to complex
- **Topological Clarity**: Avoid crossing edges when possible

### Mathematical Rigor
- **Verify Norms**: Always check norm preservation
- **Respect Dimensions**: Don't exceed 8D for persistent state
- **Formal Verification**: Use Coq proofs for critical operations
- **Property Testing**: Test algebraic properties, not just examples

### Performance Optimization
- **Polynomial Degree**: Keep degrees low for efficiency
- **Parallel Execution**: Use pure functions for concurrency
- **Memory Management**: Leverage immutable structures
- **Caching Strategy**: Cache expensive computations

### Code Quality
- **Type Safety**: Use explicit types for all interfaces
- **Error Handling**: Use Result types, not exceptions
- **Documentation**: Include mathematical rationale
- **Testing**: Comprehensive property-based tests
